## 🧠 Overview

Welcome to the **2025 version** of my personal portfolio!  
Built with 💡 simplicity, 🎨 style, and 🤖 smart features — it’s more than a portfolio, it’s a full experience.

---

## 🌟 Key Features

| 🔹 Feature                | Description                                                                 |
|--------------------------|-----------------------------------------------------------------------------|
| 🎧 Music Player           | Immersive background music with full control & sleek UI                     |
| 🤖 Badut AI Assistant     | Fun AI chatbot with voice, emoji, and interactive bubbles                   |
| 🎮 Mini Games             | Logic-based games for brain challenge lovers                                |
| 🌙 Auto Dark/Light Mode   | Automatically adapts to user system theme                                   |
| 🎯 Dynamic Hero Section   | Animated typing and scroll effects for engaging intro                       |
| 🔐 Auth Panel Slider      | Smooth panel transition for login/register (modern app-style)               |
| 🌀 Premium Preloader       | Stylish intro animation mimicking high-end systems                          |
| 📢 Smart Popup Ads        | Auto popup with sleek design and dismiss feature                            |
| 🧱 Clean Code Structure    | Modular folders for easy scaling and customization                          |

---

## 🔧 Tech Stack

- **HTML5**, **CSS3**, **JavaScript**
- Libraries: `Typed.js`, `Swiper.js`, `AOS.js`
- Extras: `LocalStorage`, `GitHub Pages` hosting
